# Split Fictionを遊んだ

友人とSplit Fictionをクリアまで遊んだ。同じスタジオが制作した前作のIt Takes Twoがゲームが得意な人から苦手な人まで楽しく遊べる非対称2人協力アクションゲームという非常に稀有なジャンルですこぶる出来が良かったので、こちらもやりたいと友人に頼んで遊んでみてもらった形。

やってみた感じ、アクションの非対称性と美しさはどちらかというと前作の方が上だったような気がするが、道案内が親切になったりキャラクターやストーリーが素直に受け入れやすいものになって更により万人受けするものになっていたと感じた。クセが減った、ってことだろうか。

ただ、前作は一応リアル世界に基づいた場所の探索が多く？ロケーションの遷移に理由付けがあったので冒険を継続している感じがあったのが、今作は設定上特に関連のない複数の「ファンタジー」「SF」世界を突破していく形となっているので、いかんせんプッツリと区切られた独立ステージという印象が強い。また前作にあった息抜きのミニゲームがより大きなサイドストーリーとして目立つ形で配置されるようになった関係でなけなしの「探索要素」にあたるものはほぼなくなっており(一応特に意味のない反応を返すオブジェや語りが聴けるベンチなどは存在するが…)本当にゴールを目指すだけになっているのもちょっと寂しいところではあった。

とはいえ大本の2人でカジュアルに協力して進んでいくアクションゲームという部分は非常に遊びやすく、誰にでもおススメできる神ゲーとなっている。仲のいい人、あるいは仲良くなりたい人と、是非。
