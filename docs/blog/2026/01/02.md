# パチスロで初めて勝ったよ

盆と正月の長期休みには、50%位の確率でパチスロ…パチンコは初回だけであとはずっとスロットなのだが…のお誘いが来る。未だに機種独自の仕様どころかスロット全体としてのルールというか定石を3割も分かっていない気がするが、とりあえず防振りのスロットがやってて楽しいことは確かだ。

特に台の選別をすることもなく、盆と正月という多分店としては設定を絞りがちな時期に行くだけなので毎度毎度俺は大量に金をスるのが恒例なのだが、なんと今日は2万円を突っ込んで7.3万円が返ってくるという+5万円の大勝ちをした。隣のお誘い人は3.5万スってていたらしいが。

今回打ったのはマギレコ。シナリオを8つ集める事こそ叶わなかったものの、金枠まどかカードのベルをぶち当てたり、まどほむそれぞれとコネクトしたり、2種類のスペシャルエピソードを見たり、2400玉を超える勝ちによるエンディングからの覚醒ATと、多分常識的な範囲内の運で見れる粗方の演出は見れた気がする。隣のプロが言うには設定としては俺の台は並だったらしいなので、ただただ豪運だっただけのようだ。

設定なんてごくまれに出る示唆演出で高設定がどうかが予測できる程度で俺みたいな素人には関係ないよなぁと思っていたが、どうやら小役？のような特段確率が低くない事象の発生率にも設定による違いはあるようで、そうだったら話は変わる。確率試行は50%に近い事象であるほど少ない回数で収束してくれるため…。今度、ちゃんと信頼区間とか使って計算してみようかな。

話は変わるが、パチスロを遊んでいて、「確率表記をゲーム内でしてくれよ！」という部分で俺はそこそこ納得がいっていない部分がある。個人的にはソシャゲのガチャしかり、宝くじしかり、くじ引きは「どれくらい当たりにくいか」が可視化されていて初めてドキドキが生まれると思っている。1%のあたりだからこそ、5回で当てたらめちゃくちゃ嬉しいし、300回で当たっても全く嬉しくないのであって、どれくらいのあたり率のくじか分からないのでは、100回の試行で当選したとしてそれが喜ばしいのか判別付かないのは当然のことだ。

なのであらゆる事象が確率試行で成り立っているパチスロは、あらゆる要素の確率を可視化させた方がプレイヤーにとって「楽しい」と思うし、表記するのは通常の設定時の値で別に構わないのでその場合メーカーや店は何も困らないと思うのだが、何故かそういった様々な確率はゲーム上で明記されず、色でなんとなく高確率だとか、何回もやってるとこの演出はどうせ信用ならないとか、そういう判断しか出来なくなっている。

全部が全部そうなら「なるほど、パチスロってのはあくまで確率は経験の中で感覚を得ていくものなのだな」と納得がいくのだが、ところがどっこい、大体の要素は公式のサイトから辿れるような情報サイトに全部、しかも設定ごとの値まで事細かに確認できるのだ。何じゃそりゃ。昔の攻略本前提のゲームか？

どうせなら一つくらいの機種は大体確率を覚えて、そんなサイトとかを見ながらじゃなくてもちゃんと自分が当てた要素のレア度をかみしめながらプレイしてみたいものである。やるなら今のところは防振りだなぁ。

余談だが、同じアニメやゲームのスロットを3時間も5時間も打っていると、アニメを1クール一気見したときのようなそのコンテンツへの強い気の持ってかれが発生する。同人誌とかあさりたくなる。俺は単純接触効果に弱いんだな…
