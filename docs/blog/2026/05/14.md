# LOLのアリーナが3VS3になったよ

知人に久しぶりにLOLに誘われたと思ったら3VS3になったアリーナへの招待だった。しばらく見ていなかったのでどこからが最近の変更なのか分からないが、ギミックやオーグメントのシステムもマイナーチェンジしている様子。

まずは1VS1のモードを出せばいいのに、なぜ3VS3で人を増やすのだろうか。完全にバランスを調整することが難しいから、最適解を出されないために変数を減らす方向には舵を切らないようにしているのか？俺はシンプルにタイマンでボコしボコされがしたいのに…

ARAMでも思うが、正直複数戦は自分が強いのかどうかが良くわからない。ARAMの場合は別に強くなろうとしてビルドをするわけでもないのでそれでもいいのだが、アリーナのように吟味してオーグメントや装備を整えて「この組み合わせならこういう強さが発揮されるんじゃ！？」と期待して行くのに、人数が多いとぐちゃぐちゃ～っとして結局そのシナジーがどれほどのものだったのかもよくわからない。上手い人なら冷静に各要素の寄与率も見て取れるのだろうが、俺には何が何だか…

3戦やって1度無敗でボロ勝ちした時も、正直なんで勝てているのか謎だった。やっぱり頭が固いおっさんにはサモリフくらいの情報量がちょうどいいのだろう…
