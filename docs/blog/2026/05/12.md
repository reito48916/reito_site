# 俺は実はローグライクとかを楽しめない側の人間なのかもしれない

俺は初代スレスパを全キャラA20まで登ってるし、シレン6は全部銅像立てたし、学マスのPレベルはMAXだし、ナイトレインはソロで深き夜の深度4までは潜った。なので、ローグライク的な「周回中に拾ったもので何とかする」ゲームは好きな方だと思っていた。

が、そんな俺がスレスパ2のアーリーアクセスに手を出して、登頂がめんどくさそうな順にA10を終わらせてあとは楽そうなサイレントとディフェクトを残して今抱いている感想が、表題だ。俺は、向いてないのかもしれない。「こういう」ゲームに。

ここでいう「こういう」というのは…つまり俺がスレスパ2のどんな面に対してこういう感情を抱いているのか…というのは、突き詰めれば「こんなん引きだけの運ゲーじゃん」という事になる。

スレスパ2のアセンションを登っていると、同じキャラで同じようなダンジョンを何回も周回するわけだが、あまりにもダンジョンが代わり映えしないせいか、アセンションの微妙な難易度変化なんかが気にならないレベルで、デッキの質だけがもろにプレイ体験を変えている感覚がある。アセンションが高かろうが低かろうが、強いカードがピック出来てデッキが強ければ楽勝で、弱いカードしかピック出来ずデッキが弱ければどうにもならないじゃんといった感覚だ。

そんなんローグライクなら当たり前やんと思うところで、じゃあどうして今までの「そういう」ゲーム…つまりは拾ったものでその周回を何とかしていくゲームではそういう感想にならなかったのかと言えば、多分シンプルに引きが弱くても失敗することが無かったからだと思う。

例えばシレンは特に縛りプレイをするわけでもないから最終ダンジョンとかでもめったに死ぬことなんてないし、学マスでも別に引きが弱いからと言って最終試験に落ちるなんてことは無い。だから、俺にとってのローグライク的なゲームの認識は「引きが悪ければその時の持ち物で何とかなる、引きがよかったらゲームにならないレベルでバカみたいな体験が出来る」ゲームだったのだ。多分初代スレスパもそうだった…のかな？覚えてないけど。それか血反吐を吐いて登ってたんだろうか。

それと比較して、ナイトレインの調整前のソロや深き夜、そして今回のスレスパ2は、引きが弱ければ、死ぬ。反省の余地なく、普通に死ぬ。シンプルにそれに無力感を感じるわけだ。

ナイトレインは極論敵の攻撃を全てかわせば勝てるわけでいくらでも自分に反省を寄せることはできるが、スレスパのようなカードゲームはもう無理だ。特に序盤は見えている情報から最適解を選ぶのはそう難しくなく、それでどうしようもないと言われたら「はぁ、そうですか」という気持ちになる。まぁ、全ての敵の出現・行動パターンを覚えているような熟練プレイヤーなら俺の回し方も最適ではないのだろうが…。

気になるのは、この世のほとんどのスレスパ民は俺と同じように、少なくとも「自分の中での最適解」を実行して敗走することもあるはずだ。その際、どういう気持ちなのだろうか。「運が悪かった」で済ませて特に省みることがないなら、その時間が無駄に思えてしまわないのだろうか。

俺は例えばフロムゲーをやっていてボスに負けた時とか、遊戯王をやっていて負けた時には、何かしら「学び」があるからその時間を突破や上達のためのステップだと思える。一方で、スレスパ2をやっていて負けたときに、特にそういった要素を感じ取れないからその時間が無駄に思えてしまうのかもしれない。普通は、そこで「こういうこともあるから、こうしなきゃな」みたいな学びがあるのだろうか。ひょっとしたら、そういう学びを初代スレスパで終えてしまっているがために、2がただその学びを辿って消化する作業になっていて、失敗するランが無駄に思えているのかも。

だとすれば、登頂した際にも上達の達成感ではなく、運がよかっただけと感じてしまい作業が終わった感覚になってしまうのも自然なことに思える。
