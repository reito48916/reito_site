# とっくに実用的な対話AIエージェントは自然言語のやり取りにとどまってないという話

俺は一応大学で自然言語も取り扱う研究室にいたこともあって、当時Transformerがどうだとか言ってた最新の真相ネットワークの技術的な話は一応理解の範疇にはある立場だった。その時点ではLLMが行う事はあくまで確率的モデルの出力であって、見せかけの知能としての言語応答だったように思えた。

その後言語処理どころか深層学習からも距離が離れ早7年ほど。普段からChatGPTにはお世話になっているが、エージェント的なAIを利用したことは無かった。先週末、友人に言われて先週末に初めてCodexというChatGPTと紐づいた？エージェントAIを使ってみて、その権限というか影響範囲の広さに驚き。検索もファイル操作もなんでもござれ。一体自然言語の入力からどういうそういったエージェント的な機能に接続しているのだろうか。

とりあえず友人が言う通り、Unityでのゲーム開発の立ち上げをCodexとともにやってみる。プロジェクトを作り、仕様や設計を壁打ちする。するとドキュメントやフォルダ構造、そしてソースコードまでファイルを作ってディレクトリ構造も作ってめちゃくちゃ書いてくれる。こりゃすごい。

まだ本チャンの「ソースコードを含めた実装が実際にUnityで動くか」までは見られていないが、仮にそれが出来ないとしてもこれだけでも仕様を固めていくステップがとても楽になるのは間違いない。とはいえ…AI君がどうやってこんなことをしているのかいまいちわかっていないのは、少し気持ち悪いような気もするが…。
