# シャニソン、実は一番最適化されたアプリゲーなのかも

千早さんが素晴らしい衣装でシャニソンに登場！というわけで、リリース時にちょっとやった以来のすっごい久しぶりにシャニソンに復帰してみた。

正直遊んだころの記憶が全くないのでどこらへんが変わっているのかよくわかっていないが、確かなのは今のシャニソンがデイリーなりイベント周回なりをするのがめちゃんこ楽であるという事だ。

プロセカで言うライブボーナスはとんでもない量をがつがつ消費してスキップやらオートやらで溶かせるし、学マスでは苦行のごときプロデュースもオートで放置できる。やりたいときに音ゲーをして、観たいときにMVを観るゲームとして割り切られている。これは素晴らしい。なんでイマイチ跳ねてないんだろう。結局ユーザーは苦行を求めるのか？

ただ音ゲー目当ての自分としては、やはりプロセカに比べるとプレイしていて納得感が薄い。スライドの判定も良くわからないし、そもそもロングノーツの入りにその継続部分から出ていくスライドが当たり前のように付随しているのが意味が分からない。すごい意地悪な一部の譜面にそういうギミックがあるならまぁ許せるのだが、さも当然ですという顔で普通に配置されているのが受け入れられない。自分だけだろうか？

とりあえず春香と千早が踊っている様子はかなり心躍るものだった。やよいのガシャは気づかないままスルーしてしまったので、早く復刻してもらってSkySeaStarsを結成したいものだ。…するよね？
