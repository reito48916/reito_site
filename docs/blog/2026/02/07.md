# オープンワールドとストーリーって相性悪いよね

コードヴェイン2をクリアした。トロコンが終わっていないのでまだまだプレイ中ではあるが、エルデンのマレニアのようなトンデモサブ要素が無い限り、後は消化試合だろう。

通してプレイしての感想だが…、後半のヌルゲーっぷりが顕著だった。これに尽きる。

一応バディは同行させると強すぎるため憑依オンリーで使っていたのだが、それにしてもちゃんと武器や再生力の強化を進めていくと、後半はザコ敵に苦戦することもなく、ボスもパターンを覚えるまでもなく適当にゴリ押ししていればこちらの体力よりあちらの体力が先に尽きるというありさまだった。

一番戦闘が燃えたのは最初のフィールド進行…水没都市をロクに探索しない状態でクリアした後、現代で殻から出てきたジョゼを倒すときで、ちっぽけな回復量の回復5回で戦うボス戦はめちゃくちゃひりつくもので、ちゃんと敵の攻撃を見切って倒すまで5,6回は死んだ。ここがピークだった。

水没都市をクリアしてその辺にある回復力強化や回復回数増加のアイテムを拾ってからは、ほとんど戦闘中に死におびえることは無くなってしまい、あとは探索を楽しむだけ、となってしまっていたのはちょっと残念かもしれない。

まぁキャラゲーである以上あんまり難しくするとターゲット層には刺さらないのかもな…とも思ったが。
