# マスターデュエルの周年豪華すぎィ！

昨日はマスターデュエルで4周年を記念して新パックやらめちゃくちゃ気合の入ったアニメーションやら盛りだくさんのコンテンツが提供されていた。

今までマスターデュエルで絵違いやサプライを目的にジェムを使う事は無かったのだが、これにはさすがの俺も敬意を表してお得なジェムをすべて購入。マスカレーナとリトルナイトの絵違いを引くまで引いた。1万ジェムくらいで済んだかな？

最近ゲームに課金する理由が、もっぱらコンテンツの入手のためではなく運営開発への感謝の念によるものになっている気がする。果たして健全なのか、不健全なのか？

肝心のデュエルに関してはろくすっぽ自分ではやってなくて毎月ギリギリシルバー1までを保ってブロンズに戻ってを繰り返すだけなのだが…。

それにしても、マスターデュエルが出たころは本当に硬派ッ！って感じでサプライも最低限しかなかったのに、女の子の声がするメイトまで出てきたことには驚きだ。あんまり遊戯王に関してはネットの声を見る機会が無いけど、反感の声もあるのだろうか？そんなやつもう残っていないか…
