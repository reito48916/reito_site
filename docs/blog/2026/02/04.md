# ブルアカ、PC版にしても辛いもんは辛い

以前はブルアカをスマホでやっていて、スマホのスペックもそこまで高くないゆえにUIの反応が悪かったりロードが長かったりしてあまりにもデイリーの消化に時間がかかるため引退してしまったのだが、少し前にブルアカのPC版が出たとの事で試しにやってみた。

すると…同じPCでやるにしてもエミュでやるのとは比べ物にならないほど反応がよく、動作が安定していて素晴らしい。スマホでやっていた時のイライラがかなり軽減され、「これが本当のブルアカだったんか！」とちょっと感動。

しかし…それにしたってデイリーコンテンツの消化に時間がかかる。部屋で絆を溜めて、スケジュールで挨拶回り、サークルを覗いて建造して戦隊シバいて対人5戦してその時々のボーナスコンテンツでスタミナを溶かして…、その上で大体の場合イベント的にバトルコンテンツが解放されているのでそれをこなす。普通に15分とかかかる。やっぱり何度見直しても、これで「デイリーが手軽！」を売りにしているのはヤバくないか…？

あと、久しぶりにPC版でやった時の素晴らしい点として、プレイヤーの無意味な選択肢をオートで流せるようになったのが感動ポイントだった。いつかのアンケートで自分が書いた点が改善されているというのはいい気分。

せめて好きなキャラの絆ストーリーくらいは読んであげたいと思う、今日この頃なのでした。
