# コードヴェイン2、あまりにもエルデンリング

コードヴェインとかいうゲームが実はソウルライクだよとの話を聞き、新作も出るという事でちゃちゃっと旧作をクリアして現在2を遊んでいるところ。

前作を遊んだときは、「あまりにもダークソウル！！」って感じの所感で、ソウルゲーをバンナムでキャラゲー化したらこうなりますよ、っといったゲームまんまだったのだが、2で今度は「あまりにもエルデンリング！！」となっている。無駄に採集できるフィールドの食材に乗り降り自在すぎる夢幻の乗り物…。あまりにもまんまである。

個人的には、バンナムのようなキャラゲーを主戦場としているゲーム企業にはどんどんこういうことをしてほしいと思っている。そりゃ斬新なゲームシステムや作りこまれた手触りが実現できるなら苦労はしないが、それが中途半端に終わるくらいならおとなしく既存の当たったゲームを模倣してキャラゲーの皮をかぶせてほしい。(それだってめちゃくちゃ大変なわけだが)

結局のところ、任天堂やフロムを始めとした「本気でゲームシステムを作っている会社」にそういった面で勝つのは難しいのだから、勝手を言えば大人しくいろんなIPのゲームをシステムはパクリでもっと気軽に出してほしいのだ。まぁ、老舗のゲーム企業はどこも大企業になってしまってそんなことも言っていられないのだろうが…。

もちろんコードヴェイン2はと言えば、あり得ないくらい先攻入力の猶予が長かったり技によって方向補正がバラバラだったりで操作の癖が強すぎるし、味方が強すぎたり少し探索をすると回復に余裕が出すぎてすぐヌルゲー化したり、バイクで適当にショートカットできる一方でフィールドが見えない壁だらけで登山は許さなかったりで探索の自発性が阻害されたりとエルデンリングに比べれば見劣りはするものの、バンナムのキャラゲーとしてみると比較的面白くクオリティも高い。これもきっとジャンルを既存のものに据えてお手本となるゲームがあったおかげだろう。バンナムに限らず、IPのキャラゲーを作るところは皆これを見習ってほしいものだ。
