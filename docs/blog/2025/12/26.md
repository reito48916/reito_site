# 祝！フル残業一週間！

先週今年分の主な仕事が終わり、今週は大体なんでもやっていい期間だった。ので、やりたいことをやるメンツの手伝いをすることになり、むしろ先週より早く会社に来てビルが閉まるギリギリまで仕事をして帰る一週間だった。

これはこれで楽しいのだが、なんか隣にも同じような状況？なのか分からないが毎日めっちゃ早く来て遅くまで残っている輩がいるので、いまいちテンションが上がらない。これがもしいつも自分しかいないような状況だったらウキウキで作業できるのだが…。これはどういう気持ちなんだろうな。

まぁ気にしてもしょうがないとは思うが。とにかく今日で今年の仕事は終わり！どうにか年末年始でSplitFFictionを協力プレイして、ナイトレインと塊魂くらいは片付けて、GOTYに輝いたなんとか33とか言うゲームを遊んだりしてみたいなぁ…
