# 遊戯王の制限改定って人の無意識的な認知変容が出る気がする

俺は遊戯王以外のTCGをロクに追っていないため他のTCGについての事情は知らないが、少なくとも遊戯王というカードゲームは、紙であれデジタルであれ、強いカードやテーマを発売する -> それを規制する -> 更に強いカードやテーマを発売する という流れで商売をしている。

個人的には別にこの流れは商売上普通の事だし当たり前の事だろうと思うのだが、新しい制限改定の発表の度に「この旧強いカードを使ってた人達は！？」「帰省するなら作るな」みたいな声が色んな媒体で多く見かける。

ここにも突っ込みどころはあるのだが、今回の本題はそこではなく、その改定の後の環境において、改定に対して手の平を返す人がいるというところだ。

強かったテーマが規制されて、その規制が足りずに次の環境でもまだ強かった場合。改定の発表時には「こんなに規制して！」と言っていた人が、「こんな規制で足りるわけないだろ！」と言っている。逆に発表時に「もっと規制しろ！」と言っていた人が、環境との嚙み合いなどもあり改定後落ちぶれたテーマに「ここまでしなくてもいいだろ！」と言っている。そういった層の数が割合として多いかは分からないが、サンプルとしていくつも観測できるのは、別にその人たちが鳥頭だとかKONAMIのアンチだとか言うんじゃなくて、無意識に認知が変わってしまっているのではないかと思う。

改定後の世界にいる自分にとって、「規制されたのにまだ強いテーマ」は今視点で見れば「強いのに規制されなかった」テーマだし、「規制されて弱くなってしまったテーマ」は「強くないのに規制されているテーマ」だ。その認識が過去の認知を歪ませて、「自分は改定前の過去の段階で、現在の状態が適切でないであろうことは予想がついていた」という意識になってくる。だから自然とそういった手のひらを返したような意見が出てしまうのだと思う。

TCGなんて娯楽だし別にそれでギャーギャー騒ぐのは構わないが(KONAMIはかわいそうだけど)、仕事とか人でそういうことをされたらたまったものではない。つくづく、人間と付き合いをするのは大変だなぁと思うのであった。
