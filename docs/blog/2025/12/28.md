# 特に何もない日

何も思いつかない日は、パッと思いついた事とかを書くようにしたいね。

今回は「可能性」という言葉の表現について。幼いころに可能性は「ある」「ない」という使い方で学んだ自分にとって、可能性を「高い」「低い」とする表現には気持ち悪さをずっと感じていた。そのため、自分が発現するときにはわざわざ「確率が高い」などと言っていた。

自分の中では、その気持ち悪さを「○○性というのは何かの性質を指す名詞であって、それがある、ないというのは自然だが、それ自体が高い、低いというのは不自然だから」という理屈付けをしていたのだが、高校の頃に「蓋然性」という言葉を知ってその説は打ち砕かれてしまった。

蓋然性はまさに「確からしさ」の性質を指す言葉で、こちらは高い、低いといった表現が正しいらしい。これも「○○性」ではあるのにだ。そうなると、可能性が高い低いと言ってはいけない理屈も自分の中ではなくなってしまう。

調べてみると確かに可能性を高低で表現するのは誤用ではあるようなのだが、個人的には言葉の誤用が否定されるのは言語として悪影響があるとき(本当は別の意味だから同じ文脈でも誤解を招くおそれがあるとか、他の言葉のルールに例外を追加する事になってしまうとか)なので、可能性のこの表現は心理的に受け入れていかなきゃな、なんて思ったものである。

まぁ、受け入れようが受け入れまいが、わざわざ他人の発言を指摘したりはしないし、自分としてはそういう微妙な表現は使わないのだが。
