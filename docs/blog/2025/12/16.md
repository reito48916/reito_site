# 初見の相手とならコミュ障にならないといったな。あれは嘘だ。

副業としてやっている開発業務に関する懇親会とかいうモノに連れていかれた。打ち合わせに付随する形で行われてしまったため断るすべがなかった。こちらの身内が4人、取引先の人が3人。7人で居酒屋で2時間といったところ。

いやーーーーー、地獄だった。画像処理がメイン、ちょっぴりAIといった分野でアカデミックと実務開発のあいの子のプロフェッショナルが学会だの企業だのの話を延々としている。まぁ何を言っているのかが分からない訳ではないのだが、別に俺は何かそういった分野で詳しいこともなければ提供できる最新の話題があるわけでもないので、ただただ話を聞いて相槌を打つだけ。幸せなことにこちらサイドの他の面々がめちゃくちゃ詳しいうえにコミュ強なので、俺が黙っていても特に何の問題もないのが救い。

とはいえ、客観的に見て「懇親会」としての役割をこの飲み会は果たしていたと思うし、これを以て「飲み会はクソ！」なんていう気はサラサラない。俺以外の6人の間では立派な飲ミュニケーションが成立していただろう。ただただ俺がこの分野の飲み会に向いてなさすぎるだけなのである。

だからといって主戦場のゲーム業界の飲み会に放り込まれてちゃんと立ち回れるかと言ったらそんなことはないのだが。やったゲーム、話題のゲームのゲーム性とかについて語り合うなら楽しいだろうが、大体会社の飲み会で展開されるのは昔のゲーム開発の話と会社の体制の話ばかり。なぁ、もっとゲームの中身について話した方が、楽しくないか？

来年の始めには新年会があるので、どうにか一緒の席になった人のゲーム観とかそういったゲーマーとしての議論が出来たらいいなと思う。

酒は飲まないぜ！
