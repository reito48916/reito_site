# ローグライト系のこう難易度ゲームって精神への負荷が高すぎないですか？皆大丈夫なんですか？

前提として、ナイトレインは神ゲーだと思う。素材をそのままにあそこまで繰り返し遊べる、しかもマルチプレイのゲームを作るのは並大抵の事じゃないだろうと思う。フロムゲーの「いうて慎重にやればそこまで操作難度的にはシビアでもないし難しくないよね」という感覚を吹き飛ばす時間制限による無限のプレッシャーはまさに新軸といっていい。

一方で、ソロでやっている自分の個人的なゲームプレイ体験としては、正直一番の感想は「疲れる！！」

数十分間脳が休まることもなく、ワンミスごとに精神が削られていく。死亡なんてしようものならザ・絶望。一方でボスまで倒せたときは、「上手くできた！」という感覚より「まぁ引きがよかったんだよな…」という感覚。正直プレイしていて気持ちのいいタイミングが少ない感触だ。

多分これは俺のエルデンリング、そしてナイトレインの理解度が低すぎて、引いた武器やアイテムの価値が良くわからないせいも大きいと思う。レア度が高いからと言って強い、美味しいわけではないため、いまいちランダム要素に対して脳汁が出ない。フィールドの配置も運がいいのか悪いのか判別付かないし…。

さて、ここまでナイトレインの感想をつらつら書いているだけだが、本題はそこではない。

俺はシューター以外のゲームが上手いので、自分にとっての「高難易度ローグライト」を遊ぶ機会がこれで初だった。シレンもスレスパも、クリアできるのは前提で、脳や精神に負荷がかかることもほとんどなかった(ローグライクは時間はかかるのだが…)。しかし、ナイトレインをやってみて、そういうランダム要素に振り回され、ゲームオーバーになってそこまでの苦労が全部パーになる、というのがかなり堪えるという事を始めて体験することとなったのだ。

こういう体験を、世のゲームプレイヤーたちはこれまで普通に受け入れて楽しんでいたのか…と思うと、信じられない。普通に辛くないですか…？

良くフロムゲーが好きとか、苦行ゲーが好きとか自己紹介すると、マゾゲーが好きなのね～なんて反応をされることがあるが、大間違い。こういったゲームくらいしか、自分のプレイヤースキルを十分に出してなお足りないようなソロゲーが無いっていうだけで、苦労をパーにされたら嫌だし、運ゲーだってやりたくない。ゲームスキルが高いソロプレイヤーに用意されているのは際限なく難易度が上がってくとか、ランダム要素でサバイバルさせるとか、そういうゲームばかり。俺は計算されたレベルデザインで簡単じゃないゲームがやりたいだけ！ゲームの開発が大規模化した結果製品寿命を延ばすためにランダム要素が濫用され緻密なレベルデザインがインディーズにしか見られなくなってきて久しいが、是非大手にも今一度そういうゲームを作ってほしいものだ。

とにかく、個人的な感覚としては、高い難しさがランダムにやってくるのって、精神的なストレスが大きい面でも、達成感が得にくいって面でも、あんまり好ましい体験につながらないんじゃないのか？と思ってしまうわけだが、世間一般にはそんなことないのだろうか？それとも、俺がシレンの理不尽に発狂する配信者を見て「いやそんなの全部ひっくるめても全然簡単やん…」と思うように、ナイトレインも「難しい配置」なんてなくて、「ナイトレインが想定する適切なスキル帯のプレイヤー」からしたらこう難易度なゲームなんかじゃないのかもしれないね。
